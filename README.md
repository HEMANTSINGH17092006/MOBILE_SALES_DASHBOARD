# 📊 Mobile Sales Dashboard Analysis (Power BI)

## 🧾 Project Overview

This project analyzes mobile sales data using **Power BI** to create an interactive dashboard.
It transforms raw data into meaningful insights for better business decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing brands and models
* Understand regional sales distribution
* Study customer behavior and transaction trends
* Build an interactive dashboard

---

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset includes:

* Sales Amount
* Quantity
* Transactions
* Brand & Mobile Model
* City / Region
* Payment Method
* Customer Ratings
* Date (Month & Day)

## 🔍 Key Insights

* Premium brands like Apple & Samsung generate high revenue
* Metro cities contribute the most to sales
* UPI & Card payments dominate transactions
* Sales trends vary across months and weekdays
* Customer ratings indicate high satisfaction
